PRE-REQUISITES: 
1. Turbo C/C++ IDE should be installed in your system.
2. Command prompt should be available for use.

HOW TO USE THE LOTTERY SCHEDULING PROGRAM
1. Run the program in the command prompt. Make sure that you are accessing the folder where the 125_mp1.c file is.
2. To run the program enter gcc 125_mp1.c -o a in the command prompt. Then enter a to run.
3. Choices are immediately shown in the interface, and run the program as you please.
